One-time setup (Cloudflare relay + Google + IBKR Flex)
  1. IBKR Flex query. Client Portal → Performance & Reports → Flex Queries. Create an Activity Flex Query with the Open Positions section, Summary level of detail, including Symbol, Quantity, Cost Basis, Currency, FX Rate To Base, Asset Category, Sub Category. Format XML. Note the Query ID.
  2. IBKR Flex token. In the same area enable the Flex Web Service and generate a token.
  3. Cloudflare Worker. Create a Worker, paste flex-relay-worker.js, set its ALLOWED_ORIGIN to your Pages origin, and add the variable ALLOWED_EMAIL (your Google address). Deploy and copy the Worker URL into the field above. The Flex token and Query ID are entered on this page, not in the Worker.
  4. Google. Enable the Sheets API, set the OAuth consent screen to External + Testing with yourself as a Test user, and create a Web application OAuth client whose Authorized JavaScript origin is your Pages origin (host only, no path). Paste the Client ID above.
  5. Host this file on GitHub Pages and open it from that URL. Paste the Spreadsheet ID (the part of the sheet URL between /d/ and /edit).
