One-time setup (Google Cloud + GitHub Pages)
  1. In Google Cloud Console → APIs & Services, enable the Google Sheets API.
  2. Configure the OAuth consent screen (External, app name, your email). Add yourself under Test users. It can stay in "Testing" mode.
  3. Create an OAuth client ID of type Web application. Under Authorized JavaScript origins, add the exact origin where this page is hosted, e.g. https://yourname.github.io (no path, no trailing slash).
  4. Copy the Client ID into the field above.
  5. Host this file on GitHub Pages (public repo on the free plan) so the origin is stable. Open it from that URL, not from a local file.
  6. Open your Google Sheet, copy the ID from the URL (the part between /d/ and /edit) into the field above.
